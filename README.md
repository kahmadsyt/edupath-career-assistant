# EduPath Career Assistant

## Judul Project

**EduPath Career Assistant: Chatbot Rekomendasi Program Studi S1 dan Roadmap Belajar untuk Siswa SMA/SMK/MA Berbasis NLP, Intent Classification, dan Recommender System**

## Deskripsi Singkat

EduPath Career Assistant adalah prototype chatbot berbasis Natural Language Processing (NLP), intent classification, dan recommender system yang dirancang untuk membantu siswa SMA/SMK/MA memperoleh rekomendasi awal program studi S1 berdasarkan minat, bakat, hobi, mata pelajaran favorit, gaya belajar, dan tujuan karier.

Project ini dikembangkan sebagai bagian dari tugas UAS mata kuliah Data Mining pada Program Magister Teknik Informatika S2.

## Latar Belakang

Banyak siswa SMA/SMK/MA menghadapi kebingungan dalam menentukan program studi S1 yang sesuai dengan minat, kemampuan, dan tujuan karier. Kesalahan dalam memilih jurusan dapat berdampak pada motivasi belajar, performa akademik, dan arah karier jangka panjang.

Project ini bertujuan membangun sistem rekomendasi awal yang dapat memberikan insight program studi, alasan rekomendasi, roadmap belajar awal, dan prospek karier berdasarkan input teks dari pengguna.

## Ruang Lingkup

Fitur utama yang direncanakan:

1. Input Bahasa Indonesia formal.
2. Input Bahasa Indonesia non-formal.
3. Input Bahasa Jawa umum.
4. Input Bahasa Sunda umum.
5. Intent classification.
6. Rekomendasi Top-3/Top-5 program studi S1.
7. Alasan rekomendasi.
8. Roadmap belajar awal.
9. Prospek karier.
10. Evaluasi model intent classification.
11. Evaluasi recommender system.
12. Prototype chatbot menggunakan Streamlit.

## Metode yang Digunakan

Project ini menggunakan pendekatan:

- CRISP-DM
- Text preprocessing
- TF-IDF Vectorization
- Intent Classification
- Machine Learning Classification
- Content-Based Recommender System
- Cosine Similarity
- Model Evaluation
- Streamlit Prototype
